# hippocampus-Turing-machine
Hippocampus model working with a semantic neural network, implementing a Turing machine.  
A detailed explanation can be found here: [Hippocampus model implementing Turing machine](https://www.linkedin.com/pulse/hippocampus-model-implementing-turing-machine-konstantine-kozachuk-336nf/)

## Build
To build the project, follow these steps:

```
mkdir build
cd build
cmake ..
cmake --build . --config Release
```
